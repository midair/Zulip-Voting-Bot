**Here is an example of a new vote:**
``` .py

```
---------------------------
**Then to vote for an option:**
private message voting bot saying:
``` .py

```

or public message:
``` .py
VotingBot Movie night  #(`Shift+Enter` for new line)
2
```
---------------------

public message:

``` .py
VotingBot Movie night  #(`Shift+Enter` for new line)
```
